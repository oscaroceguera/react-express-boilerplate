# React-Express Boilerplate

## Install Front

```shell
$ cd front
$ npm install
$ npm start
```

**Run storybook**

```shell
$ npm run storybook
```

**Stack**
* [React](https://reactjs.org/)
* [Storybook](https://storybook.js.org/)
* [Webpack](https://webpack.js.org/)
* [Css-Modules](https://github.com/css-modules/css-modules)

## Install Back

```shell
$ cd back
$ npm install
$ npm start
```

**Test**

```shell
$ npm test
```
